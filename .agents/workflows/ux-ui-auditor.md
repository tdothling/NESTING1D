# UX/UI Auditor Skill

## Objetivo
Analisar interfaces de usuário do projeto NESTING1D com foco em micro-interações, estados de hover, clareza visual, harmonia de cores, e refinar a experiência do usuário com nível de produção premium (ex: glassmorphism, temas modernos, tipografia elegante).

## Passos para Análise
1. **Identificação de Componentes Críticos**: Verifique botões, inputs, cards e layouts gerais.
2. **Análise de Cores e Contraste**: Garanta o uso de paletas profissionais (ex: Slate, Zinc, Neutral) em vez de cores cruas (ex: red-500 puro). Utilize cores HSL bem definidas se necessário.
3. **Avaliação de Espaçamento e Respiro**: Analise os paddings e margins (`p-`, `m-`, `gap-`) para garantir que os elementos não estejam sufocados.
4. **Micro-interações**: Verifique transições (`transition-all`, `duration-200`) em hovers e focus states em botões e inputs.
5. **Tipografia e Hierarquia**: Cheque o uso correto de tamanhos de fonte, pesos (font-bold vs font-medium) e contraste de texto.
6. **Recomendação Imediata**: Sugira a reescrita do código analisado aplicando práticas modernas como Shadcn UI, design systems consistentes ou Tailwind avançado.
